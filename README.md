# crawler
Crawler by selenium

记录一下自己第一个用selenium写的爬虫（大概就是用到了翻页、点击下载什么的操作，但是因为是第一个所以比较激动，记录一下）

## 设计思路：
先获取所有的文件列表（因为需要截止到某一个时间点的数据）-->检查文件中是否已经下载-->对没有下载的文件逐个按照url访问（这两个网站都有点傻，网页内嵌了url链接，所以还比较简单）
